# Weather CLI App

A command-line weather app that fetches real-time weather data for any city using the OpenWeatherMap API.

## Features

- Get current temperature and weather description for any city
- Remembers your last searched city between sessions
- Handles invalid cities and network errors gracefully

## Requirements

- Python 3.13+
- `requests` library

## Installation

1. Clone the repo:
```
git clone https://github.com/kroxy-dev/Weather_App.git
```

2. Install dependencies:
```
pip install requests
```

## Usage

```
python weather_app.py
```

You will be prompted to enter your OpenWeatherMap API key and a city name.

## API Key

Get a free API key at [openweathermap.org](https://openweathermap.org). Free tier is sufficient.
